# Portfolio — Expe Avomo

Portfolio personal de una sola página para Expe Avomo, consultor y analista junior de ciberseguridad.

## ¿De qué va esto?

Es una página web estática hecha con HTML y CSS puro (sin frameworks ni librerías externas) que funciona como tarjeta de presentación profesional. Está pensada para compartir con reclutadores o adjuntarla junto al CV.

## ¿Qué incluye?

- **Presentación** — quién es, a qué se dedica y dónde está buscando trabajo
- **Proyectos** — tres trabajos prácticos de ciberseguridad con sus herramientas y resultados
- **Skills** — tecnologías y herramientas que maneja, con barras de nivel
- **Contacto** — email, LinkedIn y GitHub

## Archivos

```
index.html   → toda la estructura de la página
styles.css   → todos los estilos y animaciones
```

## ¿Cómo se usa?

Descarga los dos archivos en la misma carpeta y abre `index.html` en el navegador. No necesita servidor ni instalar nada.

Para publicarlo online se puede subir tal cual a GitHub Pages, Netlify o cualquier hosting estático de forma gratuita.
